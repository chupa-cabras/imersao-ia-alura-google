Repositório destinados ao experimetos da Imersão Alura com IA. CC @AluraOnline, @GoogleBrasil e #imersao-ia-alura-google

## Installation
    ```shell
    python3 -m venv venv
    source venv/bin/activate or set VIRTUAL_ENV "./venv"   
    pip install -r requirements.txt
    cp .env.example .env 
   ```

   Change .env keys :-)
